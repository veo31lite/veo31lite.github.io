# Veo31Lite GitHub Pages Site

This is a lightweight static website for `veo31lite.github.io`.

## Files

- `index.html` - homepage for the `veo 3.1 lite` keyword
- `veo-3-1-lite-pricing/index.html` - pricing SEO page
- `styles.css` - shared responsive styling
- `script.js` - small static demo interactions
- `robots.txt` - crawl rules and sitemap location
- `sitemap.xml` - GitHub Pages sitemap

## Deploy to GitHub Pages

1. Push these files to the root of the `veo31lite.github.io` repository.
2. Open the repository settings on GitHub.
3. Go to **Pages**.
4. Set the source to the main branch and root folder.
5. Save and wait for GitHub Pages to publish.

The live site should be available at:

```text
https://veo31lite.github.io/
```

## Local Preview

Run a simple static server from the project root:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000/
```
